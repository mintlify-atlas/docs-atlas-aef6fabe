    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8 text-center">
    Get Karma POS running in minutes
  </p>

  <Steps>
    <Step title="Clone the Repository">
      Clone the project from GitHub and navigate to the directory:
      
      ```bash
      git clone https://github.com/Neider0608/pos-frontEnd.git
      cd pos-frontEnd
      ```
    </Step>
    
    <Step title="Install Dependencies">
      Install all required npm packages using your preferred package manager:
      
      <CodeGroup>
        ```bash npm
        npm install
        ```
        
        ```bash yarn
        yarn install
        ```
        
        ```bash pnpm
        pnpm install
        ```
      </CodeGroup>
    </Step>
    
    <Step title="Start Development Server">
      Launch the Angular development server:
      
      ```bash
      ng serve
      ```
      
      <Note>
        The application will be available at `http://localhost:4200/`. The app automatically reloads when you modify source files.
      </Note>
    </Step>
    
    <Step title="Build for Production">
      When you're ready to deploy, build the production-optimized version:
      
      ```bash
      ng build
      ```
      
      <Info>
        Build artifacts will be stored in the `dist/` directory, optimized for performance and speed.
      </Info>
    </Step>
  </Steps>
</div>
