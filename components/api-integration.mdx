## API Architecture

The application follows a **typed API integration pattern** with TypeScript interfaces for all requests and responses.

## API Response Pattern

All API endpoints return a standardized response structure:

```typescript
// src/app/components/pages/api/shared.ts:1-5
export interface ApiResponse<T> {
    code: number;      // 0 = success, non-zero = error
    message: string;   // User-friendly message
    data: T;          // Typed response data
}
```

### Response Code Convention

- `0`: Success
- `200`: Success (alternate)
- Non-zero: Error or validation failure

### Usage Example

```typescript
this.masterService.getCategories(companiaId).subscribe({
    next: (res: ApiResponse<Category[]>) => {
        if (res.code === 0 || res.code === 200) {
            this.categories = res.data || [];
        } else {
            this.messageService.add({
                severity: 'warn',
                detail: res.message
            });
        }
    },
    error: (err) => {
        this.messageService.add({
            severity: 'error',
            detail: 'Error de conexión'
        });
    }
});
```

## Date Handling

### Sending Dates to API

```typescript
getInvoices(
    startDate: Date, 
    endDate: Date, 
    companyId: number
): Observable<ApiResponse<IInvoiceClients[]>> {
    return this.http.get<ApiResponse<IInvoiceClients[]>>(
        `${this.apiUrl}pos/GetInvoicesAll/` +
        `${startDate.toISOString()}/` +
        `${endDate.toISOString()}/` +
        `${companyId}`
    );
}
```

### Parsing Dates from API

```typescript
loadInvoices() {
    this.posService.getInvoices(startDate, endDate, this.companiaId)
        .subscribe({
            next: (res) => {
                this.invoices = res.data.map(inv => ({
                    ...inv,
                    created_At: new Date(inv.created_At!)
                }));
            }
        });
}
```

## Error Handling Patterns

### Service-Level Error Handling

```typescript
loadCustomers() {
    this.masterService.getClients(this.companiaId).subscribe({
        next: (res) => {
            if (res.code === 0) {
                this.customers = res.data || [];
            } else {
                this.messageService.add({
                    severity: 'warn',
                    summary: 'Advertencia',
                    detail: res.message || 'No se pudieron cargar los datos.'
                });
            }
        },
        error: (err) => {
            this.messageService.add({
                severity: 'error',
                summary: 'Error',
                detail: 'Error de conexión con el servidor.'
            });
            console.error('API Error:', err);
        }
    });
}
```
