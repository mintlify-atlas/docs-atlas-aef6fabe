# Quick Start Guide

Get Karma POS up and running in just a few minutes. This guide will walk you through the essential steps to start using the system.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js**: Version 24.x or higher (tested with v24.13.1)
- **npm**: Comes with Node.js
- **Git**: For cloning the repository
- **Modern Browser**: Chrome, Firefox, Edge, or Safari

<Note>
Karma POS requires Node.js 24+ and Angular CLI 19+ to function properly.
</Note>

## Installation in 3 Steps

<Steps>

### Install Dependencies

First, navigate to the project directory and install all required dependencies:

```bash
npm install
```

This will install:
- Angular 19 framework and router
- PrimeNG 19 UI components
- TailwindCSS for styling
- RxJS for reactive programming
- SignalR for real-time updates
- Additional utilities (jsPDF, xlsx, crypto-js, etc.)

### Start the Development Server

Launch the development server with:

```bash
npm start
```

Or use the Angular CLI directly:

```bash
ng serve
```

The application will start at `http://localhost:4200/` and automatically reload when you make changes to the source files.

### Access the Application

Open your browser and navigate to:

```
http://localhost:4200/
```

You'll be presented with the login screen. Use your credentials to access the system.

</Steps>

## Initial Setup

On first launch, you'll need to complete the initial setup:

<Steps>

### Company Configuration

Navigate to the initial setup page to configure your company information:

- Company name and details
- Tax identification (NIT)
- Business address
- Contact information

### User Permissions

Set up user roles and permissions for your team:

- Create user accounts
- Assign module permissions (View, Create, Edit, Delete, Export)
- Configure access to:
  - Punto de Venta (POS)
  - Compras (Purchases)
  - Inventario (Inventory)
  - Proveedores (Suppliers)
  - Informes & KPIs (Reports)
  - And more...

### Inventory Setup

Configure your initial inventory:

1. **Categories**: Create product categories
2. **Warehouses**: Set up warehouse/location tracking
3. **Products**: Add your products with:
   - Product name and code
   - Barcode (optional)
   - Price and cost
   - Stock quantity
   - Category assignment
   - Unit of measure

</Steps>

## First Sale

Now you're ready to process your first sale! Here's how:

<Steps>

### Navigate to POS

From the main menu, click **"Punto de Venta"** (Point of Sale) or use the direct route:

```
http://localhost:4200/pages/pos
```

### Add Products

There are three ways to add products to the cart:

<CodeGroup>

```typescript Barcode Scanner
// Focus the barcode input field and scan
// The system automatically searches and adds the product
```

```typescript Product Code
// Type the product code in the search field
// Press Enter to add to cart
```

```typescript Product Search (F2)
// Press F2 to open the product search dialog
// Browse or search for products
// Click to add to cart
```

</CodeGroup>

<Tip>
Use keyboard shortcuts for faster operations:
- **F2**: Open product search
- **F3**: Open payment dialog
- **F4**: Process sale
- **F5**: Create new invoice
</Tip>

### Adjust Quantities

Modify product quantities directly in the cart. The system automatically:

- Validates available stock
- Reserves inventory in real-time
- Prevents overselling
- Updates totals instantly

### Process Payment (F3)

Press **F3** or click the payment button to open the payment dialog:

1. **Select Payment Method**:
   - Cash (Efectivo)
   - Card (Tarjeta)
   - Financing (Financiación)
   - Mixed payments (multiple methods)

2. **Enter Amount**: The system calculates change automatically

3. **Optional**: Select or create a customer

4. **Confirm Sale**: Press **F4** or click "Procesar Venta"

### Generate Invoice

After processing, the system:

- Creates a numbered invoice (e.g., FV-001)
- Generates a PDF invoice
- Updates inventory
- Records the transaction
- Shows the invoice preview

</Steps>

## Key Features at a Glance

### Real-Time Stock Management

Karma POS uses intelligent stock reservation:

```typescript
// When you add a product to cart
1. System checks available stock
2. Reserves the quantity for this invoice
3. Other users see reduced available stock
4. Stock is released if you remove the product
5. Stock is permanently deducted on sale completion
```

<Warning>
If you try to add more than available stock, the system will warn you and prevent the addition.
</Warning>

### Multi-Invoice Management

Handle multiple customers simultaneously:

- Press **F5** to create a new invoice tab
- Switch between invoices by clicking the tabs
- Each invoice has its own cart and customer
- Stock reservations are tracked per invoice

### Discounts

Apply discounts at two levels:

1. **Product Discount**: Individual item discounts (already configured on products)
2. **General Discount**: Apply a percentage discount to the entire invoice

## Common Workflows

### Processing a Quick Sale

1. Scan/enter product barcode
2. Adjust quantity if needed
3. Press **F3** for payment
4. Enter amount (cash)
5. Press **F4** to complete
6. Print/email invoice

### Creating a Quote

1. Add products to cart
2. Select customer
3. Navigate to **"Cotizaciones"** (Quotes)
4. Convert to invoice when ready

### Checking Inventory

1. Navigate to **"Inventario"** (Inventory)
2. View stock levels by warehouse
3. Check **"Stock Bajo"** (Low Stock) for alerts
4. Create purchase orders as needed
