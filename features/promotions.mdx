## Promotion Types

### 1. Percentage Discount

Apply a percentage discount to selected products or categories:

```typescript
const percentagePromotion: Promotion = {
  name: 'Summer Sale',
  description: '20% off all beverages',
  type: 'percentage',
  value: 20,  // 20% discount
  startDate: '2026-06-01',
  endDate: '2026-08-31',
  categoryIds: [1, 2, 3],  // Apply to specific categories
  isActive: true
};
```

<Note>
  **Validation**: Percentage values must be between 0 and 100.
</Note>

## Dynamic Form Behavior

The promotion form adapts based on the selected type:

```typescript
onPromotionTypeChange() {
  const t = this.currentPromotion.type;
  
  switch (t) {
    case 'buy_x_get_y':
      this.buyXBy = 'category';
      this.freeYBy = 'category';
      this.currentPromotion.minPurchase = undefined;
      this.currentPromotion.maxDiscount = undefined;
      if (!this.currentPromotion.buyY || this.currentPromotion.buyY < 1) {
        this.currentPromotion.buyY = 1;
      }
      break;
      
    case 'percentage':
    case 'fixed_amount':
      this.currentPromotion.freeCategoryIds = [];
      this.currentPromotion.freeProductIds = [];
      this.currentPromotion.buyY = undefined;
      break;
      
    case 'bundle':
      this.currentPromotion.minPurchase = undefined;
      this.currentPromotion.maxDiscount = undefined;
      this.currentPromotion.buyY = undefined;
      this.currentPromotion.freeCategoryIds = [];
      this.currentPromotion.freeProductIds = [];
      break;
  }
}
```

## Category and Product Selection

### Multi-select with Checkboxes

```typescript
onCategoryChange(categoryId: number, event: any) {
  if (event.checked) {
    if (!this.currentPromotion.categoryIds) {
      this.currentPromotion.categoryIds = [];
    }
    if (!this.currentPromotion.categoryIds.includes(categoryId)) {
      this.currentPromotion.categoryIds = [
        ...this.currentPromotion.categoryIds, 
        categoryId
      ];
    }
  } else {
    this.currentPromotion.categoryIds = 
      this.currentPromotion.categoryIds?.filter(c => c !== categoryId) || [];
  }
}

isCategorySelected(categoryId: number | undefined): boolean {
  if (!this.currentPromotion.categoryIds || !categoryId) return false;
  
  const normalizedCategoryId = typeof categoryId === 'string' 
    ? parseInt(categoryId, 10) 
    : Number(categoryId);
  
  return this.currentPromotion.categoryIds.some(id => {
    const normalizedId = typeof id === 'string' 
      ? parseInt(String(id), 10) 
      : Number(id);
    return !isNaN(normalizedId) && normalizedId === normalizedCategoryId;
  });
}
```

## Statistics Dashboard

Track promotion performance:

```typescript
get activePromotions(): number {
  return this.promotions.filter(p => p.isActive).length;
}

get totalUsage(): number {
  return this.promotions.reduce((sum, p) => 
    sum + (p.usageCount || 0), 0
  );
}

get expiringSoon(): number {
  return this.promotions.filter(p => {
    if (!p.endDate) return false;
    const endDate = new Date(p.endDate);
    const today = new Date();
    const diffTime = endDate.getTime() - today.getTime();
    const diffDays = Math.ceil(diffTime / (1000 * 60 * 60 * 24));
    return diffDays <= 7 && diffDays > 0;
  }).length;
}
```

## API Integration

The promotions service provides full CRUD operations:

```typescript
@Injectable({ providedIn: 'root' })
export class PromotionsService {
  private apiUrl = 'https://localhost:7197/api/';

  getPromotions(companiaId: number): Observable<ApiResponse<Promotion[]>> {
    return this.http.get<ApiResponse<Promotion[]>>(
      `${this.apiUrl}Promotion/GetPromotions/${companiaId}`
    );
  }

  getPromotionById(id: number, companiaId: number): 
    Observable<ApiResponse<Promotion[]>> {
    return this.http.get<ApiResponse<Promotion[]>>(
      `${this.apiUrl}Promotion/GetPromotionById/${id}/${companiaId}`
    );
  }

  createPromotion(model: Promotion): Observable<ApiResponse<any>> {
    return this.http.post<ApiResponse<any>>(
      `${this.apiUrl}Promotion/CreatePromotion`, 
      model
    );
  }

  updatePromotion(model: Promotion): Observable<ApiResponse<any>> {
    return this.http.put<ApiResponse<any>>(
      `${this.apiUrl}Promotion/UpdatePromotion`, 
      model
    );
  }

  deletePromotion(model: Promotion): Observable<ApiResponse<any>> {
    return this.http.request<ApiResponse<any>>(
      'delete', 
      `${this.apiUrl}Promotion/DeletePromotion`, 
      { body: model }
    );
  }
}
```

## Permission System

The module respects user permissions for the "Promociones" module:

```typescript
private applyPermissions(): void {
  const moduleName = 'Promociones';
  const permission = this.permissions.find(p => p.module === moduleName);
  
  if (!permission) {
    this.resetPermissions();
    return;
  }
  
  this.canView = permission.canView;
  this.canCreate = permission.canCreate;
  this.canEdit = permission.canEdit;
  this.canDelete = permission.canDelete;
  this.canExport = permission.canExport;
}
```

## Best Practices

1. **Always normalize IDs**: Convert string IDs to numbers for consistent comparison
2. **Validate dates**: Ensure promotion periods are valid and logical
3. **Check permissions**: Verify user permissions before allowing operations
4. **Track usage**: Monitor `usageCount` to measure promotion effectiveness
5. **Clean up on type change**: Clear irrelevant fields when promotion type changes
