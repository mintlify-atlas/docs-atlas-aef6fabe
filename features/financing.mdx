# Customer Financing

The financing module enables you to manage customer payment plans, track installment payments, and generate professional financial documentation. This feature is essential for businesses that offer credit to their customers.

### Payment Management

The system tracks financing agreements and their associated payments:

```typescript
interface Financing {
```

## Implementation Guide

<Steps>
  <Step title="Enable Financing Permissions">
    Configure user permissions for the "Financiaciones" module:
    
    ```typescript
    const permissions = {
      module: 'Financiaciones',
      canView: true,
      canCreate: true,
      canEdit: true,
      canDelete: true,
      canExport: true
    };
    ```
  </Step>

  <Step title="Load Financing Data">
    Retrieve all financing agreements for display:
    
    ```typescript
    loadFinancings(showMessage = false, number: number = 0) {
      this.posService.getFinancings().subscribe({
        next: (res) => {
          this.financings = res.data || [];
          this.filteredFinancings = [...this.financings];
        },
        error: () => {
          // Handle error
        }
      });
    }
    ```
  </Step>

  <Step title="Register a Payment">
    Add a new payment to an existing financing agreement:
    
    ```typescript
    savePayment() {
      // Validate payment amount
      const remainingBalance = this.selectedFinancing?.currentBalance || 0;
      if (this.newPayment.amount > remainingBalance) {
        // Show error: payment exceeds balance
        return;
      }

      const payload = {
        id: this.selectedFinancing.id,
        paymentAmount: this.newPayment.amount,
        paymentMethod: this.newPayment.method,
        notes: this.newPayment.notes,
        registeredByUser: 1
      };

      this.posService.addFinancingPayment(payload).subscribe({
        next: (res) => {
          // Payment registered successfully
          this.loadFinancings(false, 1);
        }
      });
    }
    ```
  </Step>

  <Step title="Generate Documents">
    Create PDF receipts and clearance certificates:
    
    ```typescript
    // Import pdfMake
    import * as pdfMakeModule from 'pdfmake/build/pdfmake';
    import * as pdfFontsModule from 'pdfmake/build/vfs_fonts';

    const pdfFonts: any = (pdfFontsModule as any).default || pdfFontsModule;
    pdfMake.vfs = pdfFonts.pdfMake?.vfs || pdfFonts.vfs;
    ```
  </Step>
</Steps>

## Document Generation

### Payment Receipt

Generate professional payment receipts with full payment history:

```typescript
generateReceipt(payment: any) {
  const docDefinition = {
    pageSize: 'A4',
    pageMargins: [40, 50, 40, 50],
    content: [
      // Header
      { text: 'MI TIENDA POS', style: 'title' },
      { text: 'Comprobante de Financiación', style: 'subtitle' },
      
      // Customer info
      {
        table: {
          body: [
            ['Cliente:', this.selectedFinancing.customerName],
            ['Factura:', this.selectedFinancing.invoiceNumber],
            ['Método de Pago:', this.getPaymentLabel(payment.paymentMethod)]
          ]
        }
      },
      
      // Payment history table
      {
        table: {
          headerRows: 1,
          body: [
            ['Fecha', 'Recibo', 'Método', 'Valor Pagado', 'Saldo Restante'],
            ...this.financingPayments.map(p => [
              new Date(p.paymentDate).toLocaleDateString('es-CO'),
              p.receiptNumber,
              this.getPaymentLabel(p.paymentMethod),
              this.formatCurrency(p.paymentAmount),
              this.formatCurrency(p.remainingBalance)
            ])
          ]
        }
      }
    ]
  };
  
  pdfMake.createPdf(docDefinition).open();
}
```

## Filtering and Search

Filter financing records by customer name or invoice number:

```typescript
applyFilters() {
  const c = this.filters.customer.toLowerCase().trim();
  const i = this.filters.invoice.toLowerCase().trim();
  
  this.filteredFinancings = this.financings.filter(f => 
    (!c || f.customerName.toLowerCase().includes(c)) && 
    (!i || f.invoiceNumber.toLowerCase().includes(i))
  );
}

resetFilters() {
  this.filters = { customer: '', invoice: '' };
  this.filteredFinancings = [...this.financings];
}
```

## Currency Formatting

The system uses Colombian Peso (COP) formatting:

```typescript
formatCurrency(value: number): string {
  return new Intl.NumberFormat('es-CO', {
    style: 'currency',
    currency: 'COP',
    minimumFractionDigits: 0
  }).format(value);
}
```

## Validation Rules

<Note>
  **Payment Validation**: The system automatically validates that:
  - Payment amount is greater than zero
  - Payment does not exceed remaining balance
  - Payment method is selected
</Note>

```typescript
if (!this.newPayment.amount || this.newPayment.amount <= 0) {
  this.messageService.add({
    severity: 'warn',
    summary: 'Campo requerido',
    detail: 'Debes ingresar un valor de abono válido.'
  });
  return;
}

const remainingBalance = this.selectedFinancing?.currentBalance || 0;
if (this.newPayment.amount > remainingBalance) {
  this.messageService.add({
    severity: 'error',
    summary: 'Monto inválido',
    detail: `El valor ingresado supera el saldo pendiente.`
  });
  return;
}
```
